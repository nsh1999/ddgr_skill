The domain names **example.com**, **example.net**, **example.org**, and **example.edu** are [second-level domain](/wiki/Network_domain "Network domain") names in the [Domain Name System](/wiki/Domain_Name_System_blocklist "Domain Name System blocklist") of the [Internet](/wiki/Internet "Internet"). They are reserved by the [Internet Assigned Numbers Authority](/wiki/Internet_Assigned_Numbers_Authority "Internet Assigned Numbers Authority") (IANA) at the direction of the [Internet](/wiki/Internet "Internet") (IETF) as [special-use domain names](/wiki/Special-use_domain_name "Special-use domain name") for documentation purposes. The domain names are used widely in books, tutorials, sample network configurations, and generally as examples for the use of domain names. The [Internet Corporation for Assigned Names and Numbers](/wiki/ICANN "ICANN") (ICANN) operates websites for these domains with content that reflects their purpose.

## Purpose

[[edit](/w/index.php?title=Example.com&action=edit&section=1 "Edit section: Purpose")]

The domains *example.com*, *example.net*, *example.org* and *example.edu* are intended for general use in any kind of documentation, such as [technical](/wiki/Technical_documentation "Technical documentation") and [software documentation](/wiki/Software_documentation "Software documentation"), manuals, and sample software configurations. Thus, [documentation writers](/wiki/Technical_writer "Technical writer") can be sure to select a domain name without creating naming conflicts if end-users try to use the sample configurations or examples verbatim. The domains may be used in documentation without prior consultation with IANA or ICANN.

In practice, these domain names are also installed in the Domain Name System with the [Internet Protocol](/wiki/Internet_Protocol "Internet Protocol") (IP) addresses for [Internet Protocol version 4](/wiki/IPv4 "IPv4") ([IPv4](/wiki/IPv4 "IPv4")) and [IPv6](/wiki/IPv6_rapid_deployment "IPv6 rapid deployment") of a [web server](/wiki/Web_server "Web server") managed by [ICANN](/wiki/ICANN "ICANN"). The domains are digitally signed using [Domain](/wiki/Domain_Name_System_blocklist "Domain Name System blocklist") (DNSSEC).

The zone files of each domain also define one subdomain name. The [third-level domain](/wiki/Third-level_domain "Third-level domain") name *www* resolves to the IP addresses of the parent domains.

## History

[[edit](/w/index.php?title=Example.com&action=edit&section=2 "Edit section: History")]

The second-level domain label *example* for the top-level domains com, net, and org have been reserved and registered since at least 1992. The IETF established the authority of this use in 1999.

In 2013, the status and purpose of the domains was restated by the IETF as belonging to a group of *special-use domain names*.

In 2025, the domain received a minor update to its appearance and content when visited over HTTP(S).
